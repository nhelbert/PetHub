どこでもホイール for Windows 98,ME & NT4.0, 2000, XP
●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●
概要

　インテリマウス非対応のアプリケーションでホイールの回転によるスクロールを可能にします。
　Word95, Excel95, メモ帳, ワードパッド など、Windows 標準のスクロールバーを使用しているアプリケーションであれば、たいてい使用できます。
　その他にも、Shift を押しながらのホイールボタンクリックや、IntelliMouse Explorerのサイドボタンを押したときに任意のキーを入力させる機能があります。



●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●
インストール（セットアップ）

　すでに旧バージョンを使用しているときは、設定ファイル(DOKOWHEL.INIなど)のバックアップをとって下さい。旧バージョンへの上書き時に dokowhel.dll のコピーに失敗するときはいったん Windows をリブートしてください。


　適当なフォルダを作成し、そこで QDOKO37.LZH を解凍（展開）してください。
　解凍（展開）の方法については http://www.vector.co.jp/for_users/guide/ をご覧ください。

以下のファイルが入っています。
READMEW .TXT	今見ているファイル
DOKOWHEL.EXE	どこでもホイール本体
DOKOWHEL.DLL	どこでもホイール本体
DOKOWHEL.HLP	どこでもホイールヘルプ
DOKOWHEL.INI	Windows 95用 INIファイル
DOKO98  .INI	Windows 98用 INIファイル
DOKOME  .INI	Windows ME用 INIファイル
DOKONT4 .INI	Windows NT4.0用 INIファイル
DOKOY2K .INI	Windows 2000用 INIファイル

●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●
インストール終了後

　エクスプローラまたは、ファイルマネージャなどから DOKOWHEL.EXE を実行してください。実行すると、システムトレーに常駐します。

　スタートアップフォルダに登録しておくと、Windows を起動と同時に実行されるので便利です。
　setupdoko.exe を実行すると、スタートアップフォルダにショートカットが作成されます。

●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●
終了

　システムトレーの「どこでもホイール」のアイコンをクリックして、メニューから「常駐解除」を選んでください。

●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●●
アンインストール

　レジストリは使用していませんので、エクスプローラでインストールしたフォルダごと削除してください。
　また、setupdoko.exe を実行した場合は、スタートアップからどこでもホイールへのショートカットを削除してください。

v3.1のバージョンからどこでもホイールを起動するＯＳによって、定義ファイル名を自動的に変更するようになりました。
	Windows 95	DOKOWHEL.INI
	Windows 98	DOKO98.INI
	Windows ME	DOKOME.INI
	Windows NT4.0	DOKONT4.INI
	Windows 2000,XP	DOKOY2K.INI
　これにより、コマンドラインで定義ファイル名を記述する必要は無くなりました。

　従来、コマンドラインに何も指定しないときは DOKOWHEL.INI が読み込まれていましたが、Windows NT4.0 では DOKONT4.INI を読み込むようになっていますので、ご注意ください。
